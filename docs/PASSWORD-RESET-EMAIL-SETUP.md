# PrePora - Custom Password Reset Email Setup Guide

## Kya Hoga
- Email body mein **"Firebase"** ka naam nahi hoga
- **"PrePora"** brand dikhega app icon ke saath
- Professional dark theme email with centered **Reset Password** button
- Button click = password reset automatically

> **Note:** Firebase apna chhota sa footer automatically add karta hai (service email ke end mein). Usko remove nahi kar sakte — but email ka poora body tumhara PrePora branded hoga.

---

## Step-by-Step Setup (Firebase Console)

### Step 1: Firebase Console mein jao
- URL: https://console.firebase.google.com/
- Project **prepora-c2d23** select karo

### Step 2: Authentication > Settings > Email Templates
1. Left sidebar mein **Authentication** pe click karo
2. Upstream **Settings** tab pe jao
3. **Email templates** section dhoondho
4. **Password reset** row pe pencil (edit) icon pe click karo

### Step 3: Sender Settings
| Field | Value |
|-------|-------|
| **Sender name** | `PrePora` |
| **Reply-to email** | `support@prepora.com` (ya apna email) |

### Step 4: HTML Template Paste Karo
1. Editor mein **HTML mode** switch karo (bottom mein toggle hoga)
2. File kholo: `docs/password-reset-email-template.html`
3. Poora HTML copy karo
4. Firebase ke editor mein **paste** karo (pura replace karo)
5. **Save changes** click karo

### Step 5: Test Karo
1. Apne app mein "Forgot Password" flow use karo
2. Apna email daalo
3. Check karo email inbox
4. PrePora branded email aayega with app icon + Reset Password button
5. Button pe click karo — password reset page khulega

---

## Available Firebase Template Variables

| Variable | Kya karta hai |
|----------|---------------|
| `%LINK%` | Password reset URL (auto-generated by Firebase) |
| `%EMAIL%` | User ka email address |
| `%DISPLAY_NAME%` | User ka display name |
| `%APP_NAME%` | App name (Firebase Console > Settings se set hota hai) |

Template mein `%LINK%` aur `%DISPLAY_NAME%` use ho raha hai.

---

## Kaise Kaam Karta Hai (Flow)

```
User "Forgot Password?" pe click
  -> Email enter karta hai
  -> Flutter: FirebaseService.sendPasswordReset(email)
  -> Firebase: sendPasswordResetEmail() call hota hai
  -> Firebase: Humara custom HTML template use hota hai
  -> PrePora branded email user ko milti hai
  -> "Reset Password" button pe click
  -> Firebase ka reset page khulta hai
  -> Naya password set karta hai
  -> Done!
```

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Default Firebase email aa raha hai | HTML mode mein paste karo, save karo |
| Logo load nahi ho raha | Supabase URL check karo, CORS issue ho sakta hai email clients mein |
| Button kaam nahi kar raha | `%LINK%` variable mat badlo — Firebase auto-replace karta hai |
| Email spam mein ja raha hai | Normal hai naye domains ke liye — SPF/DNS records setup karo |
| Firebase footer dikhta hai | Ye normal hai — Firebase automatically add karta hai, remove nahi ho sakta |
